# Talk_With_Me
Um programa que tem como funcionalidade falar o texto que você escrever e como segunda opção escrever o texto que você fala.

A ideia deste projeto é fazer com que você possa ter uma resposta em voz robótica caso precise e que tenha uma funcionalidade de poder falar e o computador digitar para você.

O intuito é ajudar tanto a ter uma resposta por voz, caso não possa falar e/ou poder falar e ter o texto pronto para poder usar em algum caso especifico.

A principio você usará o terminal para rodar o código, vai aparecer se quer digitar para o computador falar, se colocar "S; s; Sim; sim" ele vai deixar que digite para o 
computador repetir o que foi digitado. Se colocar "N; n; Não; não; nao; Nao" ele vai escutar o que quer falar e vai digita. Caso demore para falar ou fale alguma coisa que o
reconhecimento de voz não entenda o programa será finalizado.
